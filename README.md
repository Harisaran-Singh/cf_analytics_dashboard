# CF Analytics Dashboard

A web-based analytics dashboard for Codeforces users that provides insights into contest performance, rating progression, problem-solving patterns, and topic-wise strengths through interactive visualizations.

## Features

### User Profile Analytics

* Current Rating
* Maximum Rating
* Problems Solved
* Contests Participated
* Average Rating Change

### Rating Analysis

* Rating Progression Visualization
* Contest Performance Trends
* Historical Rating Tracking

### Rating Prediction

* Linear Regression-based Next Rating Prediction
* Trend Analysis using Previous Contest Data

### Problem Solving Analytics

* Most Solved Problem Tags
* Topic-wise Problem Distribution
* Difficulty-wise Problem Distribution

## Tech Stack

* React.js
* JavaScript
* Axios
* Recharts
* Codeforces API

## Installation

Clone the repository:

```bash
git clone https://github.com/Harisaran-Singh/cf_analytics_dashboard.git
cd cf_analytics_dashboard
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

## Usage

1. Enter a valid Codeforces handle.
2. Fetch user statistics using the Codeforces API.
3. Analyze rating trends, contest performance, problem-solving activity, and topic-wise strengths.
4. View predicted future rating based on historical performance.

## Dashboard Insights

* Rating Progression Over Time
* Predicted Next Rating
* Most Practiced Topics
* Difficulty Distribution of Solved Problems
* Contest Participation Statistics
